Casita de Regalos usa la información del cliente solo para cotizar, personalizar, preparar y coordinar pedidos por WhatsApp.

Uso de datos

Podemos solicitar nombre, teléfono, dirección de entrega, mensaje para tarjeta, fecha, horario, referencias visuales y datos necesarios para preparar el detalle.

  • Los datos se usan para atender la solicitud, confirmar disponibilidad, coordinar pago y gestionar la entrega.
  • No vendemos ni compartimos datos personales con terceros ajenos al pedido.
  • Las fotos de productos entregados pueden usarse para promoción, salvo que el cliente solicite lo contrario.
  • El cliente puede pedir corrección o eliminación de sus datos escribiendo al WhatsApp oficial.
